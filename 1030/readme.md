## 基于桶
最大的曼哈顿距离就是 max(r0,R-1-r0) + max(c0,C-1-c0)。

r0 表示曼哈顿原点距离矩阵左边的距离，R-1-r0 表示曼哈顿原点距离矩阵右边的距离。最大的曼哈顿距离一定是这两个值中较大的那个。
