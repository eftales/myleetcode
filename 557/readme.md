## 以特征字符(逗号空格)分词时忽略最后一个单词
557.py 的第 11 行有一句：

```python
subStrs.append(subStr)
```

这句的作用时对 subStr 做一个兜底。只执行上面的循环，如果输入字符串最后一个字符不是空格的话，会导致最后一个单词无法加入到 subStrs 里面。
